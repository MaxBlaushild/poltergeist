# House sigils

Drop the five house emblem images here, named exactly:

- spires.png    (castle + crescent moon)
- chains.png    (chain ring + blood drop)
- cinders.png   (phoenix / flame)
- ashglass.png  (hourglass + eye)
- court.png     (crown + drop)  — Marquess's Court

The app inverts them (black line-art on white → light emblem on the dark theme),
so plain black-on-white PNGs work as-is. Transparent PNGs also work.
